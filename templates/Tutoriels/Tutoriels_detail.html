À propos de ce tutoriel

Django est un framework web Python de haut niveau qui encourage un développement rapide et une conception propre et pragmatique. Dans ce cours complet, nous allons explorer les nouveautés de la version 5.1 et construire une application de A à Z.

Ce que vous allez apprendre
  • Structure d'un projet Django 5.1
  • ORM et gestion des données
  • Templates et Moteur de Rendu
  • Authentification et Sécurité

Configuration initiale

Avant de commencer, assurons-nous d'avoir un environnement virtuel prêt. Voici la commande pour installer Django 5.1 :

pip install django==5.1.0

Ensuite, créez votre premier projet avec :

django-admin startproject mysite
